## ▶ Sobre

Esta aplicação permite converter arquivos Excel para o formato OF de forma rápida e prática, diretamente pelo navegador, sem necessidade de instalação.

---

## ▶ Como usar

1. Acesse o link acima  
2. Faça upload do seu arquivo Excel  
3. Aguarde o processamento  
4. Baixe o arquivo convertido
